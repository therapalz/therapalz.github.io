Therapalz

A therapeutic object designed to help caretakers better care for individuals with Alzheimer's.




About the Project



Purpose

Therapalz aims to provide comfort that is specialized for the unique situation for Alzheimer's caretakers. Its simplistic features provides both engagement and serenity for the care receiver at an affordable scale.


Features

Right now Therapalz is in the form of a stuffed animal (pictures to the right). We've packed some electronics inside which allow for a Therapal to react to petting interactions, it purrs like a real cat! There's also a gentle amount of heat Therapalz gives off for optimized snuggling.



Under Construction

We are hard at work making Therapalz function reliably and consistently. Our next steps include deeper interaction studies with users in Spring 2016. Stay tuned!
